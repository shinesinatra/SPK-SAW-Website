# SPK SAW Website

## Deskripsi
Sistem Pendukung Keputusan berbasis website menggunakan metode SAW (Simple Additive Weighting) untuk membantu menentukan alternatif terbaik berdasarkan beberapa kriteria penilaian.

## Fitur
- Manajemen Data Kriteria
- Manajemen Data Alternatif
- Input Nilai Penilaian
- Perhitungan Metode SAW
- Ranking Hasil Keputusan

## Teknologi
- PHP
- MySQL
- Bootstrap

## Cara Menjalankan
1. Clone repository
2. Import database ke phpMyAdmin
3. Jalankan project melalui XAMPP
